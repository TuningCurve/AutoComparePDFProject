AutoPDFDiff
====


AutoPDFDiffは、修正前と修正後のPDF（またはエクセル）の差分を出力するシンプルなツールです。📑

AutoPDFDiff is a simple tool that outputs the difference between before and after PDF (or Excel) correction.

## Demo
